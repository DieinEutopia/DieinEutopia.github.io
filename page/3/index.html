$E(t)=K(t)\quad$，金融资产市场

$L(t)^{s}=L(t)^{d}=L(t)$， 劳动市场均衡 $(s$代表供给 $\operatorname{supply}, \quad d$ 代表需求 demand $)$ ；

$Y(t)=L(t) C(t)+K(t)^{\prime}+\delta K(t),$ 产品市场均衡。

$e(t)=k(t)$；

$f(k(t))=c(t)+k(t)^{\prime}+(n+g+\delta) k(t)$

$r(t)=f^{\prime}(k(t))-\delta$
$w(t)=f(k(t))-k(t) f^{\prime}(k(t))$。
（这两个是家庭最优化行为的结果）

市场经济与资本主义间关系的梳理

发表于 2019-03-22 分类于 学习笔记

以下是对市场经济与资本主义的关系的简单梳理：

市场经济是由市场配置资源的经济形式，但由于市场配置资源的经济形式，已不再是简单的商品生产，而是普遍化的商品生产。

普遍化商品生产的市场经济≠简单商品生产有 ⇒ 有市场≠市场经济。

市场经济的特点:

  • 劳动和所有权相分离
  • 社会化的雇佣劳动
  • 赢利最大化
  • 资本运动为主导

资本主义生产方式的基本特征：

一般而言在商品生产中“劳动”采取了价值形式，而在普遍化的商品生产，即所谓市场经济中，价值形式是进一步资本化了的。

商品和货币的运动 ⇒ 资本的运动和再生产过程 ⇒ 资本的循环周转过程

结论： 从生产方式和经济关系，即马政经的角度来看，所谓“市场经济”实际上就是资本主义经济。

以上内容参考了陈文通的《深度解析“市场在资源配置中起决定性作用”》（陈文通教授不愧堪称用正话反说的方式挖中特墙脚的中央党校第一人），在此做部分摘录：

市场经济本质上是商品生产的普遍化和价值形式的资本化。“市场”是一个古老的概念，是同商品交换相联系的。我们现在所说的市场是市场经济中的市场。因此，谈论市场的作用必须首先搞清楚市场经济的涵义和实质。“市场经济”这个概念来自西方经济学，产生于资本主义经济充分发展的19世纪末、20世纪初。在此之前，政治经济学中经常使用的是“交换经济”和“货币经济”。“市场经济”这个概念是由“市场”和“经济”两个词汇联结而成的，突出了“市场”现象和市场的作用，力图以此表现通过市场建立经济联系、交换产品和服务、调节利益关系和社会生产的经济形式。

市场经济中的市场是一个由资本市场主导的市场体系。市场作为商品流通过程的总表现，是在商品流通过程中形成和发展的。但是，市场经济中的商品，不同于简单商品流通中的商品，它几乎包括一切劳动产品和一切商品化了的使用价值。首先是作为商品来交换的劳动产品，包括消费资料和生产资料。其中，生产资料既是劳动产品，又是再生产的生产要素。由这些劳动产品交换而形成的市场只是一般的(普通的)商品市场。在简单商品生产时代，所交换的主要是消费品。随着社会分工和商品生产的发展，生产资料越来越成为商品；于是，商品生产进入了“用商品生产商品”的时代。其次，随着生产的社会化和小生产者被逐步淘汰，特别是随着劳动和所有权的分离，劳动力也成了商品——不同于一般商品的特殊商品，于是，劳动采取了雇佣劳动的形式，生产资料采取了资本的形式。再次，随着商品生产的进一步发展和普遍化，进入了一个全面商品化的时代。一切对生活、生产、资本增殖等等具有使用价值而又拥有所有权的东西——包括土地在内的自然资源、货币、资本、有价证券、文物、古董等等——都表现为商品了。这些都超出了本来意义的商品的范围。与此相适应，各种不同类型、不同层次的市场——劳动力市场、土地市场、房地产市场、金融市场、证券市场、猎头(人才)市场，等等——也随之出现了，从而构成了一个全方位的市场体系。在一些国家，性交易和枪支买卖等等也是合法的。这时候，处于支配地位的市场，不再是一般商品市场，而是资本市场。一切以雇佣劳动为基础、以剩余价值为目的的生产，无论是商品还是货币，无论是生产要素(物的要素和人的要素)还是生产的商品，都成为资本的要素，分别表现为货币资本、生产资本和商品资本。可见，在市场经济中，市场和市场体系本身也发生了质的变化，一般的商品市场变成了以资本要素为内容、以资本市场为主导的现代市场体系。不仅如此，资本是无孔不入的，是没有任何地域、民族和国家界限的。于是市场体系越来越成为城乡一体、各地域一体、国内外一体的市场网络。

但是，“市场”和“经济”所表达的是不同层次的经济关系。“市场”是同商品交换相联系的，经历了不同的发展阶段。在自然经济和简单商品生产时代，“市场”是各自孤立的区域性的商品交换的场所。在商品生产普遍化的现代社会，“市场”是商品流通过程的总表现，体现的是整个社会商品生产者之间的交换关系和经济关系。“经济”范畴则与此不同，它所表现的是生产方式和生产过程，例如，自然经济和资本主义经济。可见，“市场经济”这个概念没有表现出特定生产方式的性质，不能认为是一个科学合理的范畴。目前一般认为，市场经济是由市场配置资源的经济形式。但是，由市场配置资源的经济形式，已经不是简单商品生产，而是普遍化的商品生产。作为普遍化商品生产的市场经济和简单商品生产有质的区别。所以，不能认为有了市场就有了市场经济。在当今世界上，市场经济总是以劳动和所有权相分离为前提，以社会化的雇佣劳动为基础，以赢利最大化为目的的，以资本运动为主导。这些都是资本主义生产方式的基本特征.事实上，构成市场经济市场主体的企业，主要是以资本主义生产方式为基础的企业；个体生产者和共同体生产者的产品也会进入市场，但不可能成为主体。如果一般地说，在商品生产中劳动采取了价值形式，那么特殊地说，在普遍化的商品生产——市场经济——中，价值形式则进一步资本化了。商品和货币的运动同时表现为资本的运动，再生产过程同时就是资本的循环周转过程。因此，从生产方式和经济关系的角度看问题(这正是政治经济学的研究对象)，所谓的市场经济实际上就是资本主义经济。可见，市场经济不过是从交换方式和流通过程来看的资本主义经济，因而，不过是从社会经济的表面特征界定和区分经济形式的概念。所谓“社会主义市场经济”，既不是社会主义性质的市场经济，也不是社会主义社会(或社会主义条件下)的市场经济，而是中国特色社会主义道路初级阶段这种特殊社会形态的市场经济。尽管经济结构和西方国家的市场经济有所不同，但基本特征是类似的。

对市场调节作用的考察有不同的着眼点。市场经济中的经济关系是一种劳资对立的经济关系，站在不同阶级的立场上看问题，以不同的历史观看问题，对市场调节作用的考察其着眼点也不同。马克思的经济理论首先着眼于市场调节对社会生产力和工人阶级的影响，进而着眼于社会劳动按比例分配规律(即社会总劳动按照社会需要的一定比例分配于社会生产的各个部门的规律，这是人类社会生产的一般规律)借以实现的形式。在市场经济中，市场调节有利于资本积累和劳动生产率的提高，但同时导致工人的失业和贫困化、经济分化、生产相对过剩。在市场经济中，社会劳动按比例分配规律借以实现的形式，不是社会的直接分配和有意识的调节，而是交换价值。因此，这个一般规律也就采取了价值规律的形式。西方经济学则把市场经济视为人类社会的一般经济形式，从而认为，经济学的基本问题是：生产什么，如何生产(以何种方式方法生产)，为谁生产，生产多少(这些基本问题在任何生产方式和社会形式中都存在)。由于大多数经济资源(生产要素)存在稀缺性，因此认为，对市场调节作用的着眼点是资源配置的效率和优劣。实际上，资源配置的效率和优劣，完全是相对于资本而言的。只有在市场之外的再分配领域，才谈得上所谓的“公平”。

市场调节包含着对经济利益的重新调整和再分配。商品价格、资本利息、平均利润率、个别利润率、地租(以及租金)、工资和工资率、股票价格等等的变动，都会导致新价值的再分配，导致剩余价值和利润的重新分割。市场对利益关系的调节主要涉及三类关系：一是劳资关系以及劳动者的内部关系(就业和工资)；二是资本之间的关系(个别资本地位的改变，资本的优胜劣汰和你死我活，资本的联合、兼并和重组)；三是剩余价值(利润)在有关当事人(资本所有者，企业主，土地所有者，以及服务阶级)之间的分配和再分配关系。这些利益关系不过是市场经济关系(主要是在资本主义生产方式基础上形成的关系)的表现和反映。这些利益关系同时表现为阶级关系和阶层关系。在现有的生产方式和经济关系范围内，通过这种调节，达到经济利益的相对平衡和社会关系的相对“和谐”。市场对社会生产的调节或市场决定资源配置，就是在这种经济关系和利益关系的基础上进行的。市场调节的过程，就是资本竞争和追逐利润的过程，就是资本和劳动的不断运动和优胜劣汰的过程，就是资本剥削雇佣劳动的过程。正因为如此，市场调节必然带来部分工人失业、经济分化、相对贫困(在一定的阶段上还存在绝对贫困)、生产的相对过剩乃至经济危机(信用危机或金融危机归根到底是经济危机的表现和结果)等等。

……

市场调节是一定经济关系的表现
市场的调节作用是通过价值形式和市场机制进行的，但归根到底是商品形式和市场主体的作用。既然市场是在商品交换中形成的，是商品流通过程的总表现，那么，市场的作用就归结为市场主体的作用，而市场主体的作用又基于市场主体的独立自主性质。市场主体无非是商品的生产者和购买者(包括直接的消费者和商人)。商品交换和商品流通之所以能够形成，取决于市场主体在经济上是否独立自主。一方面，所有市场主体(首先是商品生产者)在经济上是独立的，他们是商品的所有者，他们处在社会分工的体系之中而互相依赖，所生产的产品不仅具有使用价值，而且具有交换价值；另一方面，所有市场主体能够自主地生产、运输、出售和购买具有交换价值的产品。在市场经济中，生产、经营、投资主体实际上是资本的人格化，因此，市场主体的独立自主同时就是资本运动的独立自主。相对于一般的市场主体而言，资本主体的独立性就更加重要；没有这种独立性，就没有真正的竞争，也没有对自己行为的约束力，合理的资源配置就是一句空话。

假设市场存在许多家庭，数量为H，且也都是同质家庭，则可由一个代表性的家庭代表所有家庭。家庭在其预算约束下最大化自身效用的行为由下式给出：

$m a x \int _ { t = 0 } ^ { \infty } e ^ { - p t } u ( C ( t ) \frac { L ^ { s } ( t ) } { H } ) d t$

s.t.

$C(t) \frac{L^{s}(t)}{H}+\left(\frac{E(t)}{H}\right)^{\prime}$

$= r(t) \frac{E(t)}{H}+W(t) \frac{L^{s}(t)}{H}+\frac{\pi(t)}{H}$

其中，

$u ( C ( t ) ) = \frac { C ( t ) ^ { 1 - 0 } - 1 } { 1 - \theta } , \theta \neq 1$ ;

$u ( C ( t ) ) = \ln C ( t ) , \theta = 1$

这是一个常相对风险规避效用函数（CRRA），规避系数为 $\theta$。$C ( t )$，是单位资本消费，$\rho \in(0, \infty)$ 是时间偏好率，即贴现率。$E(t)$ 是经济中的总的金融资产，初始值为 $E(0)$ 给定, $\pi(t)=0$。

定义 $c(t) \equiv \frac{C(t)}{A(t)}, e(t) \equiv \frac{E(t)}{A(t) L(t)}$ 分别为单位有效劳动的消费，单位有效劳动的金融资产; $B(t) \equiv \frac{A(0)^{1-\theta} L(0)}{H}, \beta \equiv \rho-n-(1-\theta) g$ 可以将上述最优化式子转化为下述简便的形式:

$\max \int_{t=0}^{\infty} e^{-\beta t} B \frac{c(t)^{1-\theta}}{1-\theta} d t$

s.t. $e(t)=[r(t)-n-g] e(t)+w(t)-c(t), \quad e(0)$ 给定

根据第二篇文章中的知识点，c(t)是控制变量，e(t)是状态变量，这个最优化问题是一个无穷边界自治问题，tvc是 $\lim _{t \rightarrow \infty} e(t)=e^{*}$，

定义hamiltonian function：

$H(t, c(t), e(t)$,

$\lambda(t))=e^{-\beta t} B \frac{c(t)^{1-\theta}}{1-\theta}$
$+\lambda(t)[((r(t)-n-g) e(t)+w(t)-c(t)]$ ,

$\lambda$ 是现值朗格朗日乘子

$\frac{\partial H}{\partial c}=0: e^{-\beta t} B c(t)^{-\theta}=\lambda(t)$ -(2)
$\frac{\partial H}{\partial e}=-\lambda^{\prime}(t): \lambda(t)[r(t)-n-g]=-\lambda^{\prime}(t)$ -(3)
$\frac{\partial H}{\partial \lambda}=e^{\prime}(t): e^{\prime}(t)$
$=e(t)[r(t)-n-g]+w(t)-c(t)-(4)$

对（2）式两侧取对数，再对t求导，联合（3）式，得：

$\frac{e^{\prime}(t)}{e(t)}=\frac{1}{\theta}[r(t)-(\rho+\theta g)]-(5)$

那么，式子（4）和式子（5）构成了两个非线性动态系统，边界条件是tvc和e(0)。

Foreign units in Rojava

In 2014, an ISIS-embattled YPG (People’s Protection Units) called for foreign volunteers to join in their fight against black-flagged fighters from Syria and beyond. The initial volunteers for the Kurdish struggle included foreign fighters sympathetic to the Kurdistan Workers’ Party (PKK), trained militia from Matthew VanDyke’s Sons of Liberty International (a not-for-profit business that trains Christian militias in northern Iraq), and a slew of other Western veterans. A large portion of the initial Western veterans appeared to harbor religious (Christian) motivations. After it became clear, however, that the YPG was dominated by cadres of Marxists, Marxists-Leninists, and Maoists, this flow of western veterans has stemmed, and was replaced by a new wave of foreign volunteers who wished to join the cause.

Through a website set up by the YPG, English-speaking foreign fighters were able to contact the Kurdish group, begin learning the local Kurdish dialect, Kurmanji, and book travel to YPG territory. The route to Rojava ran through Suleimaniya, Iraq for years until the Kurdish Regional Government (KRG) of Iraq took action to stem the flow of fighters. The YPG recruitment site is now just a landing page explaining that the route is blocked, but the remainder of the old site is still accessible through unlinked pages.

IFB subunits in Rojava

The International Freedom Battalion (IFB) is a foreign militia harkening back to the internationalist brigades in the Spanish Civil War. Members of the IFB come from many nations, including the UK, the US, Germany, Greece, and France. Subunits include communist, anti-fascist, and anarchist Western elements as well as Turkish communist parties of varying ideologies.

BÖG subunits in Rojava

The largest group within the IFB is the Birleşik Özgürlük Güçleri (United Freedom Forces, BÖG). BÖG, like IFB, contains a mix of foreign brigades that espouse leftist political ideologies. BÖG also has a women’s unit, the Kadin Özgürlük Gücü (Women’s Freedom Forces, KÖG) and a number of Turkey-based Kurdish Communist groups.

The BÖG was founded at the end of 2014 during the battle for Kobanî. The IFB was founded in the middle of 2015 by partisans of the BÖG in coalition with several other leftist pro- Kurdish groups that had previously not joined BÖG. BÖG keeps ties with the PKK in Turkey.

HBDH subunits in Rojava

Another coalition of pro-Kurdish groups, the Halkların Birleşik Devrim Hareketi (Peoples’ United Revolutionary Movement, HBDH) contains exclusively Turkish communist groups. HBDH battalions include Stalinist, anarchist, Marxist-Leninists, and Leninist organizations. The Türkiye Komünist Partisi/Marksist-Leninist (Communist Party of Turkey/Marxist-Leninist, TKP/ML) is a political party that operates within Rojava and has created an armed wing called the Türkiye İşci ve Köylü Kurtuluş Ordus (Liberation Army of the Workers and Peasants of Turkey, TIKKO).

Western brigades of IFB

Among the IFB are units that are explicitly comprised of volunteers from Western countries. These include Επαναστατικός Σύνδεσμος Διεθνιστικής Αλληλεγγύης (Revolutionary Union for Internationalist Solidarity, ΕΣΔΑ). Εσδα is a Greek Anarcho-Communist group in Rojava. The Antifascist Internationalist Tabûr (International Antifascist Battalion, AIT) is made up of Western members from a multitude of countries, led by an Italian. Brigade Henri Krasucki (Henri Krasucki Brigade, BHK) is a French volunteer force named after a French labor rights activist. BHK is modeled after the Bob Crow Brigade, a British volunteer force also named after a labor rights activist. All four of these brigades have pledged allegiance to IFB and coordinate directly with the YPG in Rojava. The remainder of the foreign volunteer forces are from Turkey.

Conclusion

Compared to the YPG, foreign volunteers fighting on behalf of the YPG (especially Western volunteers) remain few in number. For now, foreign volunteers represent a set of poorly-trained but ideologically-willing fighters and a significant international media boon for the Syrian Kurdish political cause.

自然也有风险，运气不好就要坐牢杀头、麻袋沉江。然而民国本来就是一个走在大街上都会吃流弹的神奇社会，左圈人士只要不真下基层做反贼，收益总要大于危险：混得好了，还有被官府收编来反左的希望，什么行政院长、中央宣传部长，还不是手到擒来；倘若时局有重大变化，又可以进步人士面目坐享厚禄。岂不美哉？

有一个青年，是山东官僚大地主的后裔。少年时期，曾在中学里参加自干五，一不小心被官办的五毛党吸收了，大有被侯聚森的味道。可是他不曾想到共青团公众号发财，而是来到大上海寻求真理。

青年一到上海，就得不到家里的资助，穷困潦倒了。幸好青年混了左圈，总还不缺二三朋友帮助。其中，一个朋友写左圈自嘲小说，大体内容无非是左圈青年整天撕逼、百无一用、被“直男农民”拒之门外云云，现已发达，在出版界混出了名堂。于是，借助在朋友那里，靠给商务印书馆点校古籍为生。不过青年的古文实在不太通，竟至错谬百出；七扣八扣，连生活费都紧张了。

生活如此窘迫，幸好青年情商不错。不几时，竟已同多位左圈大老谈笑风生。左圈大老们正在搞国防文学，邀青年赞助。青年受到提携，自觉遇到伯乐，自然不吝效力。于是，青年在左圈渐渐有了名气，投稿也有人登、生活也逐渐宽裕了。

此时正有另一青年，从满洲打游击回来，著小说一部，行销四方。青年看后大受感动，也蹭个热点，写作书评一篇，惋惜其不能在满洲多打一阵游击、多体验体验生活。孰料此文被鲁迅看见，联想到青年跟一众左圈大佬在上海吃吃喝喝、天天装逼、大搞国防文学、玩弄概念翼赞国府，却好意思指责满洲的游击战士离开战场，不由作《三月的租界》怒喷之。

青年跟出版界的朋友从外面归来，在朋友的朋友那里看到鲁迅文章的校样。作为鲁粉，青年当即如晴天霹雳五雷轰顶，久之才自语要去信解释。然而树欲静而风不止，青年的朋友中却颇有几个打抱不平的，于是论战又过了一阵才消停下去。不久鲁迅去世，青年悲痛万分，写下不少纪念文章。

八一三淞沪抗战爆发，青年就与左圈的大佬和朋友们分道扬镳，北上去济南带走了年长的弟弟妹妹，奔向敌后参加抗战。在日寇的铁壁合围之中，青年终于也成了一个游击队员、一名共产党人，而鲁迅保护的学生却躲到延安消沉去了。

很多年之后，成了中年的青年回到了上海，成了文化战线的高级干部。而左圈大佬们也纷纷回到了上海，只是他们已经无法再在一起相互提携了。

又过了很多年，成了老年的青年坐在监狱里，看到左圈大佬的徒子徒孙用上海滩阴谋小说的笔法，将一个身无分文标点古书度日的青年说成是领津贴的文化特务，又把他从朋友的朋友那里偶然看到鲁迅文章校稿描写成是特务奉命搜查印刷厂的“罪证”，不禁莞尔一笑。

后来青年死了。做国民党特务已经不再是社会上感到可耻的事情，于是他又在蜥蜴人笔下成了共济会特派员。与此同时，新一代左圈大佬则忙于发明他的“法庭讲话”、“狱中家书”。相信他要是看到这一幕，也是会发笑的吧。

另补：

这位到了中年的青年到了上海以后，被上海的官方大佬看中提拔，仕途还算不错，当然那个时期有一批左圈大佬沉浸于放卫星，小土炉，进口丝袜糖果换货币中，这位青年和那位提拔他的大佬向某人写了一篇文章，被某人赏识。

当然好境不长，大佬在某个事件前昔在某地突然去世，这位少年也受到排挤。当然仅仅过了一年，风云突变，上海也成了斗争的中心，青年看到如此形势，立即准备了某个组织，把革命群众团结出来，可是上海的左圈大佬和他们手下的喽啰并不买账，要保住自己的乌纱帽，想尽各种办法，最后迫不得已想出了一个“狠招”钞票，想利用钱来压制革命，可惜被青年识破，马上抛出反对经济主义，要求政治挂帅的口号，次年一月夺权，成立上海公社，青年被推为领袖，可惜左革命大佬又不买账，说公社是无政府云云，当然上海公社也成为革委会，左圈大佬依旧不买账，贴炮轰青年的大字报，说什么革命队伍不纯，老干部少，把某个组织排斥出去云云，过了几个月算是把这些人压制下去。

军队里那几个左圈大佬看不惯青年，找青年的岔，成为二号人物以后，开始向青年发难，在江西某地，几个左圈大佬轮流炮轰青年，头号人物不愿意，批评了那几个左圈大佬，后来某个黑纲领，青年又成了头号目标。

军队里那几个左圈大佬倒台后，青年正式进入中央，被某人物器重，当然左圈人士又开始鼓噪起来，什么整顿，什么四化，什么提高国民经济，什么条条专政，什么三项指示为纲，什么台阶论，青年不以为然，写了基本理论著作来反驳之，青年整理的某人著作也被左圈大佬乱改一通被中央大佬打回去了(左圈把那本书当作圣经的不少)青年起草的人大报告左圈大佬为了给自己邀功大面积删改报告，在以后的多次会议左圈大佬对于青年敌视找茬，到了某事件，左圈大佬和他们的迷弟贴大字报炮轰青年。

当青年死后，左圈大佬纷纷后悔，用各种理由搪塞，否认往事，称赞起青年和“支持”起文革来了，青年看见了也会笑吧。

Groups that make up the Syrian Democratic Forces

TRANSNATIONAL:

KCK

(Koma Civakên Kurdistan / Group of Communities in Kurdistan)
Leader: Abdullah Öcalan aka Apo
Established in 2007 to promote the ideology of Democratic Confederalism formulated by Öcalan following his arrest in 1999. Functions as a transnational umbrella organization for the various organizations linked to the PKK, in theory it functions as the supreme authority but in practice the majority of political power remains in the hands of the PKK leadership

KJK

(Komalên Jinên Kurdistan / Kurdistan Communities of Women)
The sister organization to the KCK, the KJK was established to function as an umbrella organization for the various regional women’s rights groups affiliated with the PKK.

SYRIA / ROJAVA:

PYD

(Partiya Yekîtiya Demokrat / Democratic Union Party)
Leader: Salih Muslim
Founded in 2003 following the PKK withdrawl from Syria in the late 1990’s, the PYD is the Syrian arm of the KCK. The dominant political force in Syrian Kurdistan, it exerts a controlling influence over the YPG and other Kurdish self defence militias.

TEV-DEM

(Tevgera Civaka Demokratîk/ Movement For a Democratic Soceity)
Coalition of the PYD and various co-idelogues established to govern the Kurdish regions in Syria following the Regime withdrawal. Unilaterally established the three autonomous Kurdish cantons in Syria. Dominated by the PYD in practice.

KDPS

(Partiya Demokrat a Kurdistanê li Sûriyê / Kurdistan Democratic Part of Syria)
Leader: Abdel Hakim Bashar
Syrian affiliate to Massoud Barzani’s KDP. Initially did not join the SNC as they wanted a guarantee of Kurdish cultural rights as a prerequisite to their participation in the opposition (which was not forthcoming), however they still favour joining the SNC and oppose the system of autonomous cantons established by TEV-DEM. Ideologically hostile to the PYD.

ENKS

(Encûmena Niştimanî ya Kurdî li Sûriyê / Kurdish National Council) - Also known as the KNC
An umbrella organization uniting the KDPS and various smaller Kurdish parties midwifed by Massoud Barzani following their withdrawal from the NCC. Do not participate in TEV-DEM’s system of autonomous cantons and favour integration in the SNC with the caveat of upfront guarantees for Kurdish rights

DBK

(Desteya Bilind a Kurd / Kurdish Supreme Committee) - Also known as the KSC
Established in 2012, the DBK is a joint governing body composed of the PYD and the KNC created with the intention of administrating the Syrian Kurdish regions following the Regime withdrawal. In practice the DBK is defunct following in-fighting between the two factions who each argued that the other was seeking to marginalise it’s influence. Exists in name only but has no power or influence.

YPG

(Yekîneyên Parastina Gel / People’s Protection Units)
Established in 2004 as the armed wing of the PYD, the YPG was officially brought under the control of the DBK following it’s creation in 2012 and now functions as the armed forces of the Supreme committee. In practice the YPG is still controlled by the PYD and has squeezed out all other armed Kurdish militias in the region. YPG units have also been deployed in Northern Iraq to assist the KRG.
Estimated Strength: 50,000 - 65,000

YPJ

(Yekîneyên Parastina Jinê / Women’s Protection Units)
The all female sister organization to the YPG, some sources cite it as a brigade within the YPG while others label it a separate organization - the establishment of separate Women’s groups is a common PKK/KCK practice and there is a similar lack of clarity concerning their official status.
Estimated Strength: ~7,500

Asayişa Rojavê

(Security)
Simply called Asayia (or Asayish), this is the internal police force established by the DBK. Again, in practice it is dominated by the PYD but also works closely with Assyrian groups in the region. Not to be confused with the KRG Asayish.
Estimated strength: ~4,000

Jabhat Al-Akrad

(Liwa’ Jabhat al-‘Akrad l-Nusrah Shaʿbnā al-Sūrī / Kurdish Front Brigade to Protect the Syrian People)
Formerly a brigade within the FSA composed of Kurdish and Arab defectors from the Syrian Army, Jabhat Al-Akrad was expelled from the FSA military council due to it’s close co-operation with the PYD. Predominately active in Aleppo and Raqqa.
Estimated strength: ~7,000 (claimed)

SUP

(Syriac Union Party)
An opposition Assyrian party mainly active in Al-Hasakah, it’s armed wing the Syriac Military Council (MFS) formerly integrated into the YPG in 2014

Sutoro

(Syriac Security Office)
An Assyrian self defence force formed to protect Assyrian neighbourhoods in al-Hasakah, they function as a police force and maintain internal security. Co-operate closely with the Kurdish Asayişa and have a tacit alliance with the YPJ.

Sootoro

(Syriac Protection Office)
A Qamishli based splinter faction from the SUP, the Sootoro perform the same function as the SUP’s Sutoro but have aligned with the regime as opposed to the YPG

IRAQ / BAŞUR

KRG

(The Kurdistan Regional Government)
President: Massoud Barzani (KDP)
Prime minister: Nechirvan Barzani (KDP)
The government of the autonomous Kurdish region of Iraq

KDP

(Partîya Demokrata Kurdistanê / Kurdish Democratic Party)
Leader: Massoud Barzani
Founded by the legendary Kurdish resistance fighter Mustafa Barzani, the KDP grew to become one of the most influential Kurdish parties in the region. Led by the Barzani clan, one of the most powerful tribal groups in Northern Iraq that has weilded significant influence since the late 19th century. Perceived as being prone to corruption and driven by tribalist concerns, favours the expansion of the KRG into disputed territories considered historical Kurdish. It’s forces and predominately deployed in it’s traditional power base in the West of Iraqi Kurdistan centred around Duhok and Erbil
Estimated strength of KDP peshmerga: ~25,000

PUK

(Yekêtiy Niştîmaniy Kurdistan / Patriotic Union of Kurdistan)
Leader: Jalal Talabani
Founded by left wings figures in the KDP dissatisfied by the tribal and conservative approach of the KDP, the PUK is predominately based in the East of the Iraqi Kurdistan centred around Sulaymaniyah and draws support from a largely metropolitan base. Traditional rivals of the KDP, have a better relationship with the PKK and have developed closer ties with Iran to counter the KDP’s developing relationship with Turkey. Deploys it’s forces in the east of Iraqi Kurdistan and controls Kirkuk, a stronghold of PUK support.
Estimated strength of PUK peshmerga: ~25,000

Gorran

(Bizûtinewey‌ Gorran / Movement For Change)
A party formed to protest the corruption evident in the KRG, PDK and PUK. Gorran has steadily developed to become the second largest party in the region and has drawn a substantial number of votes away from the PUK in Sulaimaniyah and other areas traditionally under it’s sphere of influence.

Unifed Peshmerga

Forces controlled by the Ministry of Peshmerga, these exclude party militias under the direct control of each party as indicated above.
Estimated strength: ~50,000 (Professional), 100,000 - 120,000 (volunteer)

Zeravani

A highly trained militarised police force under control of the KRG Ministry of the Interior (held by the KDP). In practice the Zeravani are loyal to the KDP and are one of the best equipped and trained units at their disposal. Initially a part of the Iraqi Federal police, much of their equipment and training was funded by the central government. Have been deployed in the East of Iraqi Kurdistan and have played a highly visible role in front-line operations against ISIS.
Estimated strength: ~25,000

Asayish

(Security)
The internal KRG security agency, the Asayish are responsible for combating internal terrorism and sabotage and have conducted raids against suspected ISIS militants within KRG controlled territory.

Dizha Tiror

(Anti-Terror) - Sometimes simply called DT
An anti-terrorim special forces unit under the control of the Asayish that is in practice loyal to the PUK. Commanded by Lahur Talabani, son of Jalal Talabani. DT is primarily composed of veterans of the PUK anti-terrorism forces trained by the CIA/US Special Forces for Operation Viking Hammer.

YBŞ

(Yekineyên Berxwedana Şengalê / Sinjar Resistance Units)
A Yazi
di militia established in Sinjar following the YPG intervention in the region to secure an escape route for trapped civilians. Primarily trained and armed by the YPG, they have recently stoked controversy in the region by backing an autonomous Yazidi canton centred around Sinjar.
Estimated strength: >1,000

HPS

(Hêza Parastina Şingal / Sinjar Defence Forces)
A Yazi self defence force led by Qasim Shesho, a renowned Yazidi elder. Independent of the KRG Peshmerga but co-operates closely with them in operations and has received arms and training.
Estimated strength: >2,000

NPF

(Nineveh Plain Forces)
A new Assyrian militia under the KRG Peshmerga command structure set-up to counter the recently created NPU. Assyrians in Nineveh are divided over the future of the region with some favouring integrating the area into the KRG to protect minority rights while others are suspicious of joining the KRG and favour remaining outside the region with some pressing the central government for complete regional autonomy
Estimated strength: ????

Dwekh Nawsha

Another Assyrian self defence milita based in the Nineveh Plain formed by the Assyrian Patriotic Party.
Estimated strength: ~250

NPU

(Nineveh Plain Units)
A new Assyrian militia created by the largest Assyrian party, the Assyrian Democratic Movement (or Zowaa), and partially funded by the Assyrian diaspora. Unaffiliated with the KRG and more openly hostile to the idea of integrating Nineveh into the KRG.
Estimated strength: ????

PÇDK

(Partî Çareserî Dîmukratî Kurdistan / Kurdistan Democratic Solution Party)
The Iraqi arm of the KCK. Tiny and powerless. Has been routinely suppressed by the KDP.
Estimated strength:I don’t know - maybe like one guy or something?

TURKEY / BAKUR

PKK

(Partiya Karkerên Kurdistani / Kurdistan Workers Party)
Leader: Murat Karayılan
Commies turned Democratic Confederalists. Officially they have a ceasefire with the Turkish government but running tit-for-tat attacks persist and the PKK have begun to mobilize more openly in Turkish Kurdistan as tensions have begun to escalate. Leadership is based in the Qandil Mountains in Iraq.

HPG

(Hêza Parastina Gel / Peoples Defence Forces)
The armed wing of the PKK. Spread between it’s bases in the Qandil Mountains, Turkey, Syrian Kurdistan and has units deployed in Sinjar, Kirkuk and elsewhere in Eastern Iraq including Mexmur and Gwer which are near a large refugee camp housing PKK fighters and supporters relocated from Turkey
Estimated Strength: 5,000 - 15,000 (recruitment has swelled in recent months and reliable figures are unavailable)

YJA-STAR

(Yekîneyên Jinên Azad ên Star/ Free Womens Units)
The all female branch of the HPG.
Estimated Strength: I dunno

YDG-H

(Yurtsever Devrimci Gençlik Hareketi / The Patriotic Revolutionary Youth Movement)
The youth wing of the PKK. Have become increasingly prominent in street protests in recent months and have taken a lead in anti-government riots, some have theorised that the PKK may be losing control of an increasingly radical youth movement that has grown tired with the slow rate of the peace process. Something to keep an eye on

TAK

(Teyrênbazê Azadiya Kurdistan / Kurdistan Freedom Hawks)
Suspected Leader: Bahoz Erdal
A hardline splinter faction of the PKK responsible for a high-profile bombing campaign targeting tourist destinations in 2006. They claim to be a completely independent organization but some commentators believe it to be a front used by the PKK for more questionable acts, a true splinter faction made of former members or an alias used by active members of the PKK displeased with the peace process. I think the latter is the most likely and a resurgence of TAK is possible if the hardline faction attempts to reassert themselves
Estimated strength: Like 12 guys probably.

BDP

(Barış ve Demokrasi Partisi / Peace and Democracy Party)
The largest Kurdish party in Turkey, the BDP officially calls for an end to the PKK’s armed campaign and has played go-between for Ocalan and the PKK leadership in Iraq. Although officially not affiliated with the PKK and much more moderate than them policy wise, the BDP does draw support from PKK supporters and from former-supporters alienated by the PKK’s totalitarian style of leadership. Has transformed into a purely local government party after the formation of the HDP.

HDP

(Halkların Demokratik Partisi / Peoples’ Democratic Party)
Leader: Selahattin Demirtaş
A left-wing national political party formed from various Socialist groups and moderate Kurdish parties including the BDP. Champions of minority rights and democratic socialism - increasingly holding the balance of power in Turkish politics.

Hüda Par

(Hür Dava Partisi / Free Cause Party)
The successor party to the Islamist group Kurdish Hizbollah, Hüda Par argue for Kurdish cultural rights but have a long-running violent rivalry with the PKK (who they see as Godless communists). In the spotlight recently after clashes between the YDG-H and Hüda Par in South East Turkey.

IRAN / Rojhilat

PJAK

(Partiya Jiyana Azad a Kurdistanê / Party of Free Life of Kurdistan).
Iranian KCK affiliate. Largely based in Iraq’s Qandil mountains though it maintains an active presence in Iran. Seemed to have a cease-fire with Iran negotiated after the capture of senior PKK leader Murat Karayılan who was subsequently released, this may now be broken as there have been inconsistent reports of recommencement of hostilities.

YRK

(Yekineyên Rojhilata Kurdistan / East Kurdistan Units)
Recently re-branded armed wing of PJAK. Unknown number of members in Iran, a couple of units deployed through Iraq and Syria assisting the PKK and YPG (specifically Kirkuk). PJAK/YRK flags are rare and it can be hard to identify their presence, Iran has applied pressure on the KRG to prevent the display of PJAK/YRK symbols to prevent them gaining legitimacy from resisting ISIS.
Estimated strength: ~2,000

HPJ

(Hêza Parastina Jinê / Women’s Defence Force)
Come on, guess

KODAR

(Komelgay Dîmokratîk û Azadî Rojhelatî Kurdistan / Organisation of Free and Democratic Society for East Kurdistan)
An newly established political sub-unit of PJAK with the purpose of promoting Democratic Confederalism and working towards a negotiated settlement with Iran. Don’t know much about these folks yet.

PDKI/KDPI

(Partî Dêmokiratî Kurdistanî Êran / Democratic Party of Iranian Kurdistan)
The largest and oldest Iranian Kurdish party founded in 1946 by Qazi Muhammad. Briefly controlled the short-lived Republic of Mahabad, one of the more successful attempts at Kurdish autonomy. The PDKI leadership (and a majority of it’s membership) is in exile in Northern Iraq but a small fighting force does seem to be maintained across the border.
Estimated Strength: <5,000

KOMALA

(Komalay Shoreshgeri Zahmatkeshani Kurdistani Iran / Organization of Revolutionary Toilers of Iranian Kurdistan)
The Kurdish branch of the Communist Party of Iran, the second most popular Iranian Kurd party. Again, the parties leadership are in exile in Northern Iraq. Formerly staunch Marxist-Leninist they have drifted to the centre in recent years but still remain of the left, challenging the PDKI’s more conservative stance.
Estimated Strength: ?

PAK

(Parti Azadi Kurdistan / Kurdistan Freedom Party)
A party so small that people often forget they exist. Not to be confused with two Syrian parties with the exact same name. An Iranian group party that has been in exile since the end of the Iran / Iraq war, based in Erbil in Northern Iraq. Frequently get mistaken for PJAK but are totally different dudes. Have a nice logo.

菲律宾共产党是怎样的一股政治势力？

原为水来火去2015年在知乎上的回答，介绍较为详尽，备之

菲律宾共产党成立于1930年11月7日（其建立得到了南洋共产党（中共南洋分支）和美共等的帮助），1931年五一节因组织游行而遭美国殖民当局镇压，指导部被破坏。次年菲共被禁。1936年以后接受共产国际倡导的“人民阵线”，积极开展反法西斯运动（这时原先被捕的菲共党员也获释出狱），1938年与菲律宾社会党合并。二战期间日本占领菲律宾，菲共遂组织“菲律宾人民抗日军”（胡克巴拉赫普），积极开展抗日游击战争，一度发展到10余万之众。不过在美国重返菲律宾前后，拉瓦兄弟（他们在1930年代初期党的指导部被捕后就实际把持了党的领导岗位）控制的菲共领导集团受白劳德主义影响，奉行右倾机会主义路线，致力于走议会道路，幻想进政府做官，后来在当局的镇压下不得不重新拿起武器，把过去的人民抗日军战士组织起来建立“菲律宾人民解放军”。但之后又陷入“左”倾机会主义的速胜论，提出要在两年内取得胜利，结果反而受到重大损失。之后菲共又再度转向右倾机会主义路线，在1955年以后基本上放弃了武装斗争（只有少量部队坚持了下来，但又受到掌握军权的塔鲁克一苏木隆集团的流寇主义的影响），并实际解散了党的基层组织。

进入1960年代以后，拉瓦兄弟把持的菲共领导层不得不提拔西松等青年干部，以重建党的基层组织和发展党领导下的工运、农运、青年运动和其他群众运动。西松等人为此进行了积极的工作并取得了成效，并由此在党内建立了威信。另一方面，随着1963年中苏论战的爆发，西松等人支持中共，与支持苏共二十大和二十二大路线的党内修正主义领导集团矛盾激化。1967年4月西松率领党内革命派退出菲共，并发起“第一次整风运动”为建党作准备，新菲共于1968年10月26日正式成立。

这里顺便一提，按照西松本人于2006年为庆祝文革爆发四十周年而撰写的《无产阶级文化大革命对菲律宾共产党的影响》一文，“在文革的最初6个月内，我们恰巧在中国。我们同中央党校的中国同志进行了长时间的讨论（包括在菲律宾有长期革命经历的中国同志），主要是关于历史经验和中国、菲律宾无产阶级及人民斗争。后来，1968年10月26日，我们在马克思列宁主义毛泽东思想的指导下重建了菲律宾共产党。种种迹象表明，文革给了我们灵感并鼓励我们重建菲律宾共产党。但同时我们谨记，第一次伟大的整风运动和重建菲律宾共产党是根据菲律宾历史和环境作出的。”

通常所说的菲共，一般是指1968年底成立的、从老菲共中分离出来的新菲共。老菲共一般被称为“1930年党”，马科斯上台后的1974年因支持当局而合法化（其吹捧马科斯使菲律宾工业化，并解决了土地问题云云），此前农村的原“菲律宾人民解放军”残部也在苏木隆率领下向当局投降。如今的老菲共在群众中没什么影响力，目前确实是“基本可以忽略不计的政治势力”，不过新菲共可不是。

新菲共成立后，在反修的同时也批判了军事上的流寇主义，同时与吕宋岛农村残存的对修正主义路线不满的红色指战员取得了联系，于1969年3月29日正式建立了党领导的武装力量“新人民军”（最初是与一支只有几十人的小队伍），并由此重新开始了武装斗争。此后菲共领导下的武装斗争在曲折中前进（尽管西松本人在1977年在地下活动中被捕，新人民军司令丹特也被捕，不过菲共和新人民军总的来说还是有所发展）。进入1980年代以后，新人民军曾经发展到近3万之众。党领导下的各种群众运动也发展迅速。

不过需要指出的是，菲共自己对1980年代的发展评价不高。认为当时虽然组织力量发展起来了，但路线是错误的（1980年代的菲共在国际上一度继续与改开后的天朝保持联系，直到这种联系最后被天朝方面主动切断。同时也开始把苏联看成社会主义国家——而此前的菲共则追随毛泽东时代的中共揭批苏修全面复辟资本主义、是社会帝国主义的世界两霸之一。就菲律宾国内情况而言，当时菲共党内占上风的路线否认菲律宾的半殖民地半封建社会性质（认为菲律宾已经工业化、城市化而成为资本主义社会），因而也否认农村包围城市、持久人民战争的路线，强调以城市为中心的武装起义。后来这一路线受挫后，持此主张者又悲观动摇，转向失败主义，最后公开背叛菲共。）

旧菲共领导的武装力量叫“菲律宾人民解放军”。（“菲律宾人民解放军”是二战后菲共领导的武装力量的名称。）另，蒂诺．德．蒙多(苏木隆)暗杀了披德罗．塔鲁克及其几名亲信，然后向当局投降。

1986年马科斯倒台，阿基诺夫人上台后释放了西松、丹特等人（不过之后西松就一直流亡荷兰迄今，当然目前的菲共总的来说还是遵循西松的路线，因此西松可以说是菲共的精神领袖），并与菲共谈判（1987年在军方未遂政变后谈判中断，当然后来菲共与历届菲律宾当局的谈判始终在断断续续的进行，不过一直没有实质性的结果罢了）。不过此后不久随着谈判破裂，菲律宾当局又重新加强了对菲共的镇压，1980年代后期至1990年代初期是菲共的艰难岁月，新人民军兵力一度下降到只剩三千人。另一方面其指导部也损失很大，在此期间党内也发生了分裂（西松等出狱后与此时的菲共领导层发生了激烈的路线斗争）。这一时期，原先接替丹特出任新人民军总司令的罗幕洛·金塔纳尔也于1992年被捕后叛变，后来勾结当局、并组织反革命武装与菲共作战（此外其还受到在任期间贪污革命经费的指控），于2003年初在马尼拉被菲共方面派出的锄奸队处决（这是后话了）。

1992年西松为代表的一批老党员发起“第二次整风运动”，推动菲共重回毛派立场，重申以马列毛主义为指导思想，强调菲律宾仍然是半殖民地半封建社会，坚持以农村包围城市为基本战略、持久的人民战争为基本斗争方式。将各种机会主义分子清除出党（这些人在此前后也自立门户，建立了劳工党（社民）、棉兰老革命工人党（托派）等组织，他们攻击奉行西松路线的菲共为“斯大林主义者”，其中有些后来甚至有过与菲共方面发生武装摩擦的记录），同时在思想政治领域严厉批判现代修正主义、托洛茨基主义等等各种思潮。也是在此前后，菲共正式与天朝划清界限，并与（由于对秘鲁共产党（光辉道路）鼓吹“贡萨罗思想”不满而退出革命国际主义运动（RIM）的）德国马列主义党等联合，共同发起了RIM之外又一个主要的毛派国际组织“马列主义政党及组织(毛主义)国际大会”（ICMLPO）。整风运动后菲共及其领导的各种群众运动逐步得到恢复和发展，新人民军的总兵力也回升至一万余人，并在全国90%的地区都建立了“游击战线”。“游击战线”内的村庄普遍实行了减租减息、雇工增资、提高农产品价格等，并且向当地群众提供了免费医疗等服务，建立了各种群众组织和政权（受其直接领导的有组织的农民群众就有几百万之众），在条件许可的局部地区还进行了土改（譬如新人民军曾于2009年在尼格罗岛袭击了当地大地主、马科斯密友和阿罗约家族支持者Danding Cojuangco的庄园，这次袭击本身是在尼格罗岛进行的革命土改运动的一个组成部分，而这次运动中将没收Cojuangco家族名下5000多公顷的地产，无偿分配给当地农民和农业工人）。除此之外，菲共在城市中也领导着相当规模的工人运动、贫民运动、学生运动。

（2014年2月20日西松在接受同属毛派的巴西共产党重建联盟主办的“新文化杂志”采访时曾指出，“革命的力量和人民正在进行真正的土地改革，以便把落后的农村转变成政治、经济、社会文化方面的革命堡垒。尽管受到敌人的军事镇压，由于在共产党领导下的武装斗争和广大群众基地的建立相结合，所以革命运动变得越来越强大。现在，红色政权已经存在于拥有数千百万人民的110多个游击区，遍布在全国81个省中的71个省。”）

菲共领导着名为“全国民主阵线”的统战组织（成立于1973年4月24日），其下属的群众组织有：工会革命委员会、全国农民协会、爱国青年组织、新妇女爱国运动、科迪勒拉人民民主阵线、基督教全国解放阵线、爱国教师协会、人民艺术家作家协会、爱国政府雇员协会、爱国卫生协会、人民科学工作者联盟、劳工组织联盟、人民律师委员会、摩洛革命组织、卢马德革命组织等。此外，菲共还在2000年12月组织了用于参加选举的合法马甲“人民优先党”。

菲律宾革命者直言不讳地声称：目前菲律宾存在着两个政权，两个国家。一个是美帝走狗、代表着大买办、大地主的“菲律宾共和国”；另一个是菲共领导的人民革命政权，代表着未来的“菲律宾人民共和国”的雏形。

迄今为止菲共仍然在菲律宾境内坚持着武装斗争，通过小规模部队灵活机动的游击战打击敌军。2008年建党40周年时，菲共表示目前是其发展得最好的时期（大意如此）。另外，近年来菲律宾当局确实不断加强对菲共和新人民军的镇压力度，围剿计划层出不穷（譬如近期开展的代号为“百雅尼汉（Ba ya ni han）作战计划”的围剿行动）。不过按菲共方面的说法，当局的围剿并未阻止革命武装斗争的扩大发展，“菲律宾革命运动的前景和计划是：从目前的战略防御阶段推进到战略相持阶段。”（西松语）

事实上，新人民军在进入本世纪以后的战斗中也不乏“亮点”，试列举几例如下：2007年4月7日出动一个排奇袭棉兰老岛Davao监狱，制服守卫后“带走了监狱军火库中所有105件长武器、2件短武器、两部无线电台、众多弹药”，2009年2月15日比科尔半岛的一次伏击战中击毙政府军精锐的“侦查游骑兵”部队12人（这支部队当时还有四名美军陪同，不过然并卵）；2010年3月6日，在民都洛岛的一次伏击中毙伤敌人19名，从而一举消灭政府军一个排；2013年10月21日在北哥打巴托省对菲律宾陆军第38和第57步兵营的伏击战中毙敌9名；等等。

驳2014年菲共主席被捕说：

这无非是指去年3月的蒂亚姆松夫妇被捕事件。其实按照负责与菲律宾当局谈判的全国民主阵线（菲共领导的统战组织）和平小组主席路易斯·哈兰多尼的说法，这对夫妇只不过“是全国民主阵线的顾问，在菲政府与全国民主阵线的和谈中执行非常重要的任务。他们受全国民主阵线与菲政府签订的《关于安全和豁免保证的联合协议》的保护，逮捕他们是违反协议的行为。“（当然菲律宾当局方面抓人后安的罪名是非法持有枪枝和爆炸物，声称“在拘捕行动期间，在该对夫妇身上找到了4把枪和2枚手榴弹。”云云——非法持枪在菲律宾可保释，持有爆炸物则不能。）

当然，罗织罪名拘捕作为谈判代表的包括蒂亚姆松夫妇在内的数名菲共中央委员后（这对菲共确实是一个损失，但完全是承受得起的损失，而且这种刷下限的行径只会进一步激怒菲共），菲律宾当局确实厚着脸皮对外声称抓到了菲共主席和总书记（其宣称被捕者中的丈夫贝尼托·蒂亚姆松是菲共现任主席，其妻威尔玛·蒂亚姆松则是菲共“秘书长”），但这不过是吹牛。其中贝尼托·蒂亚姆松在1980年代末期确实一度出任过菲共主席，不过这其实也是老黄历了……事实上1992年底菲共搞二次整风时党主席就已经是一个名为阿曼德·Liwanag的人了。这对夫妇被捕后，菲共一方面指责当局手段的下作，一方面也表示其队伍中可接替他们的人很多。从事后的情况来看，菲共指导部也并未因此瓦解（菲共目前指导部的具体构成本身也是迷雾重重的，不过真要是党主席、总书记，也会这么轻易就中招么？）

至于“现菲共游击队人数有四千余”云云本身也是菲当局公布的数字（菲律宾当局一直声称新人民军兵力在减少），菲共自己的说法似乎是其领导的新人民军的兵力一直稳定在一万人左右的规模（此外还有约三万民兵）。

不可否认新人民军在近年来的战斗中确实受到一定的损失（譬如2015年6月28日棉兰老岛的新人民军领导人Leonardo Pitao的牺牲），但这些损失并不足以使菲律宾革命受挫（譬如也是在棉兰老岛，2015年间菲律宾政府军在与新人民军的作战中死亡人数超过一个营的规模）。

顺便一提，新人民军会向外国资本在菲开设的企业（包括天朝企业在内，事实上菲共认为当代天朝已经背叛了毛路线，复辟了资本主义，近年来甚至演变为列强）征收“革命税“（有时也会武力打击勾结菲律宾反动当局横行霸道、危害当地群众，或破坏环境，或拒交革命税等的外资企业、矿山、种植园等），估计这就是某答主所谓“敲诈”吧？不过么，这个某答主口中的“打着牌子没有信仰的恐怖组织”的纪律性是哪怕是被收革命税的天朝资本家也认可的：

“从2010年开始，政府部门向中国企业要钱的少了，但其他武装组织并没有放过中资企业。在马尼拉地区以外做项目时，中资企业面临的最常见风险就是被新人民军索要“革命税”。

“寓军于民”的新人民军，是菲律宾和美国黑名单上的恐怖组织。但在不得不与其打过交道的中国企业看来，其行事还算规矩。事实上，中资企业雇用的许多菲律宾雇员很可能就是他们的人。一般的程序是，他们会先让手下把项目负责人的姓名、办公地点、项目金额一一打听清楚，然后把一封有当地指挥官签字的正式信件，派人偷偷送到项目办公室。

“革命税”税额一般是项目总额的半个点到1个点之间。接到信的企业必须主动联系新人民军，他们会再次约好接头方式。周昊的一个同事就曾经跟新人民军的人见过面：眼睛要先被蒙上，被带上车后四处转几个小时，才拉到山里跟新人民军的人谈判，谈好了再想办法把钱送过去。

不给武装组织交钱的后果通常都很严重。中国水电、青岛建工等公司在南部棉兰老岛施工时，都被不知名的武装组织炸过设备。相比较来说，新人民军还算客气，“至少不会杀人，拿了钱后还会发个完税证明”。”——《中国企业“活”在菲律宾》：（南方周末 - 中国企业“活”在菲律宾））

另外，菲共党、政、军的收入来源大致如下：

“始终为菲律宾共产党、新人民军和政治权利机构开展的管理、行动和社会项目保证充足的经济财政资源。这主要有三种来源:

  1. 人民军队有为自身需求进行的生产项目。群众组织也进行这种为人民军队服务的项目。
  2. 土地改革和合作项目收入的适当比例可以志愿提供给NPA。

除了拒缴革命税的中资企业，新人民军还攻击过瑞士资本控制下的矿山，以及日资矿山和种植园。（此外还有部分来自群众自发捐款）

换言之，按照菲共方面自己的看法，征收革命税是“合理合法”的收税行为，武力打击拒缴革命税的企业则是在打击抗税行为。

另外顺便一提，革命税的征收对象也不仅仅局限于外资，本国资本也是一样。此外新人民军也同样对祸害群众的买办企业进行武力打击，譬如新人民军尼格罗岛部队根据省革命政府指令，曾在2008年9月9日袭击了买办资本的Tamlang Valley农业发展公司，烧毁拖拉机三辆（该公司是前参议员Herminio Teves的产业，大肆强占土地，“破坏当地粮食生产，改种专供国外市场用作生物燃料的木薯等作物”，导致了许多农民流离失所，同时也威胁到当地粮食安全）。

录以备考：从2003年开始，菲律宾反动当局便与国内一些敌视菲共的“左翼”政党（社民派的民主行动党、托派的革命工人党等）勾结，指控菲共有所谓暗杀名单、搞政治暗杀，国际托派等也掀起谣言攻势，污蔑菲共对不同左翼组织的领导人、积极分子实施肉体消灭。

2007年８月，在菲律宾当局策划下，这些泛左向荷兰当局指控西松同志涉嫌谋杀两个被新人民军击毙的菲共叛徒出身的反革命分子（其中之一就是那个罗幕洛·金塔纳尔），荷兰警方逮捕了流亡在荷兰的西松同志，监禁了他17天，并对他的住所和民族民主阵线在当地的办公室进行了野蛮搜查。随后因荷兰海牙地方法院认定证据不足而不得不释放西松同志。但荷兰检察官当时表示将继续搜集证据、追查此案。国际进步势力对西松同志进行了声援。经过旷日持久的斗争，2009年3月31日，荷兰全国检察官办公室正式通知西松同志，因为缺乏充分的证据，正式撤销对其“谋杀罪”的指控。这标志着帝国主义和菲律宾反动当局策划的这场针对西松同志的阴谋最终被挫败。

菲共对此认为是美帝、阿罗约政权和荷兰当局勾结策划的阴谋，并明确发表声明称——“新人民军完全对金塔那和塔巴拉两人的死亡负责。新人民军处决他们，绝不是因为他们离开了菲共，也绝不是因为他们的政治观点与菲共不同，而是因为他们犯下了具体的、严重的反革命罪行。他们贪污革命经费，和反动当局合作、出卖过去的同志，导致多名革命者被捕、被杀，甚至亲自组织反革命武装与新人民军作战。人民革命法庭认定他们犯下重罪，新人民军奉令逮捕他们，他们武装拒捕，才在战斗中被击毙。菲共说，假如反动政权和其他反革命宗派想要为他们报仇，就请直接来和新人民军作战，不要对当时早已离开菲共和新人民军实际决策岗位的西松搞阴谋。”

菲共转向者于1994年建立了所谓“人民共产党”，1998年正式建立了托派党“菲律宾革命工人党”及其武装力量“革命无产者军”，但该党在1999年就与当局进行谈判“并于2000年达成一项和平协议”。其中部分人“认为这是一种绝对而又简单的投降”，于是2001年5月1日自立门户建立“棉兰老革命工人党”和“革命人民军”，并在2003年成为第四国际(统一书记处派)的菲律宾支部。不过这个新建的托派党“在目前的形势下不考虑进行进攻性的武装斗争。因此他们在2005年与政府签订了停火协议并于2006年达成协议继续停火。”同时在与菲律宾政府的和谈中“要求政府承诺向棉兰老革命工人党影响的地区的发展项目提供更多的财政扶持，且开展什么项目应该由当地人来决定。”。

材料存放：1950年代老菲共领导的菲律宾人民解放军是在拉蒙·麦格赛赛任国防部长和总统期间基本摧毁的（当然这和老菲共自身的错误路线也有很大关系）。不过具讽刺意味的是，他的一个外孙罗赛特却在进入本世纪以后成为了支持新菲共的毛主义者：

我如何成为毛派？——对一位在美国的菲律宾留学生的采访

红色参考编辑部普罗采访

2011年7月13日

【红色参考编者按】罗赛特是一名在美国攻读博士的菲律宾青年，他和很多菲律宾革命青年一样都是坚定的马列毛主义者。值得一提的是，罗赛特出身于菲律宾大地主大资产阶级家庭，而他在长时间摸索之后决定背叛自己的阶级出身。红色参考的记者普罗特别采访了他，让国内的读者从他的经历上进一步理解菲律宾人民的觉醒和斗争。

红色参考（以下简称红）：我发现你桌上摆放着几本书，其中一本的标题引起了我的好奇：《美国在心中（America Is in the Heart）》，为什么一个毛派会读这本书呢？这本书讲的是什么？美国梦？

罗赛特（以下简称罗）：恰恰相反，这本书讲的是美国梦的反面。作者卡洛斯 布洛桑是一位伟大的菲律宾作家，他讲述了菲律宾移民在美国的悲惨生活；你知道美国总是给人一些承诺，比如自由民主平等，这些移民工人也正是追寻这些而来，但是他们所经历的恰恰是相反的东西。这和我在美国从事的社会活动也很有关系，很多从第三世界来的移民都在美国从事收入最低也最没有保障的工作，我和其他的同志一直为这些群体做维权工作。

红：说到美国，菲律宾遭受过两个殖民势力的统治，先是西班牙，到了19世纪末期又成了美国。长时间的殖民给菲律宾人民留下了深深的印记，你的名字就明显的来自西班牙语，你的英语也说的非常的标准。这样的背景对你的身份认同产生了什么影响吗？

罗：这个影响是深刻的，住在菲律宾大小岛屿上的人本来就来自很多不同的民族，历史也很悠久，殖民者有意的毁掉了很多菲律宾自己的历史记录，让我们更难回溯到殖民时代之前的历史。不过菲律宾人民的反帝国主义的斗争从来没有停止，在美西战争结束之后的1899年，菲律宾就和试图控制菲律宾的美国打了一仗，但是损失惨重的输掉了。列宁也曾特意提到了菲律宾这场很早的反帝斗争。

我们从小受的教育也是殖民化的，没有主体性的。当然我不知道现在小孩们学些什么，但是我读书的时候，历史课本里面菲律宾的历史完全是消极被动的，先是印尼人和马来人到这里来定居，之后又加入了中国移民和其他国家的人。再后来西班牙人做头头了，接着换成了美国主子。你看，这个里面没有阶级，没有斗争，菲律宾和菲律宾人在这里面永远是被动的接受的一方，谈不上什么身份意识，也没有指出什么方向。

1998年的时候，我读了康斯坦迪诺重写的菲律宾历史：《过往的延续（Continuing Past）》。这本书是头一本左派写的菲律宾历史，尽管不能说是马克思主义的，但是也是用一种人民的史观，用阶级斗争的角度来解释菲律宾的历史过程。这本书对我影响很大，而且这也正是我慢慢思想左转的时候。

红：你这个时候有多大？为什么会左转？这和你的家庭背景有关吗？

罗：我这个时候15岁。我从小就对政治感兴趣，但是并没有和任何实际的斗争结合起来。可以说我从小就自然的有反对帝国主义的情绪，我痛恨这个社会上有毒品，贫困和卖淫，菲律宾有很多这样的丑恶的现象。但是从一开始我就只是一种抽象的反对，我并没有真正的深刻的去理解这些现象的根源。

在我简单的民族主义情绪比较高的时候，我坚定的想要去参军保卫祖国，但是不久就放弃了，我不喜欢军队的环境。这是在高中的时候，同时我的思想慢慢往左转了。起先是因为我喜欢艺术，所以加入了一个马尼拉的剧社，你知道很多作家是左派，在那里我接触到了很多左翼的作品，反映现实阶级斗争的。我自己就表演和导演过其中一些。在这个过程中，我变得在情感上倾向于左派了。

我的家庭没有能给我什么思想上的帮助，但是他们的确给了我上流社会的生活条件。我母亲是一个大地主，她的家族一直是保守的望族，比如我的外祖父Magasaysay就曾经担任菲律宾的总统，他在任期里基本上摧毁了菲律宾的共产党力量。我父亲这边则是新兴的精英，我的祖父曾是菲律宾的最高法院法官，我父亲也是律师，现在则是某种交易人，专门给外国资本在菲律宾投资做中间人，简单的说就是买办。所以我的确有买办资产阶级的家庭背景，不过出于种种原因，他们从来没有工夫来管孩子的思想，所以我也没有受到什么限制。

但是我真正成为左派，还不是在剧社，而是在实际的运动中。埃斯特拉达在菲律宾执政时，对穆斯林采取了歧视的政策。尽管我们家传统上都是天主教的，我哥哥自己选择成了穆斯林，有一天我们在街上散步的时候，警察就过来搜查他了，当然他们不知道我们的背景了。我受到了直接的触动，因为以前反动的国家机器还没有侵犯到我头上来。之后我就加入了反对追查伊斯兰的社会运动中，在运动中我结识了真正的左派，并开始独立思考问题，参加左派的学习读书小组，思想转变比较大了。

但是真正让我成为一个坚定的革命者，还是在反对埃斯特拉达的运动中，埃斯特拉达不光是巨额贪污，他统治下的社会充满了贫困，破坏和战争，我当时在念高中，但是已经开始做组织工作，在马尼拉各个高中串联。我是在这个斗争过程中，变得革命的。其实就是一个阶级立场的转变过程。

红：那你是怎么成为一个毛派的呢？

罗：其实我在处于模糊的民族主义的阶段时，不喜欢马克思列宁毛泽东，我觉得那些都是外国的意识形态输出而已。但是在学习小组里面，我读到的第一本书就是菲律宾语版的《共产党宣言》，一下子就吸引了我。我读的第二本就是毛主席的《实践论》，还是菲律宾语的，当然后来我接着又读了很多马列毛的著作。你读毛主席论著的时候，就会强烈的感受到他说的那些问题和历史条件就是菲律宾的现实，而他论述的解决方法则马上可以拿到菲律宾来用。简单的说，我一读到毛主席的著作，就自然的觉得“这就是对的！”。

而且菲律宾的毛派革命运动相对比较成熟了，有理论，有组织，有纲领，有根据地，虽然现在形势比较艰巨，但是一个左翼知识分子成为一个毛派从思想上说是很自然的。

红：说说你在成为毛派之后的经历。

罗：我在组织活动的时候“出了情况”，不得不暂时离开菲律宾到美国来读大学。我开始非常沮丧，不过后来我意识到，在哪里都可以搞革命工作。我开始和这边来自各个国家的同志们一起工作。在这个过程里，我对毛主席思想的理解又更深了。到了美国的环境里，我才发觉自己在菲律宾的实践斗争中学到了很多斗争经验，比如说：为人民服务，“人民”是谁？只有在实践中，我才深切的感觉到“人民”，我是在为了谁而斗争。这就和美国的左派大不一样，很多美国的进步人士对于“人民”的理解是空洞抽象的，他们是从书上读来的，没有什么实在的阶级感情，这样的理解容易变得教条，甚至是走向反面，你看他们的运动斗争往往是没有什么力量的。

红：你是说成熟的革命运动本身会生产出成熟的革命者和理论。

罗：对，这些中心国家的左派找不到群众基础，也没有什么力量。事实上，在组织活动的时候看的很明显，第三世界来的左派有很多共识，很多问题对于我们来说早就不是问题，但是美国的左派就不是这样，我们并不生活在一个语境下面。

不过美国也给我很多锻炼。在这里没有成熟的运动，那么毛派应该怎么发挥自己的力量？难道去寻找理想的革命群众吗？现实里的群众是被分割的，是有各种情况和条件的，他们不一定是革命的，但是我们应该把他们带动起来。要做到这点，你得去深入的了解他们面临的矛盾和困难，你得从各个方面去帮助他们，这样就能建立起信任，才能动员他们，这才是毛主席的群众路线。而且毛主席说的正确处理人民内部矛盾的方法也正是我们处理不同组织之间关系的准则。

红：这也让我们回到了最开头说的你为移民工人长期做维权工作。在更广泛的意义上，说到毛派的任务，毛主席很反对机械的历史观，强调人的能动性，比如我们毛派不能坐等革命客观条件的到来，而是要去主动创造革命的主观条件，比如组织和理论。

罗：对！这是毛主席很伟大的地方。这也正是我毕业之后回菲律宾要参与的工作——建造革命的主观条件。说到这里，我深深感谢毛主席，没有毛主席的思想，菲律宾的革命运动绝不是这个样子的。

红：咱们每一代人都继承了上一代革命者的遗产，毛主席是他那一代里走在最前面的人。对了，你背的书包上面有毛主席的书法“为人民服务”，你知道毛主席是个大诗人，也是个大书法家吗？毛主席的诗词总给人无穷的力量。最后一个问题：你觉得革命客观条件什么时候会比较成熟？

罗：我真希望我懂汉语，这样就可以读他的诗词了。至于书法，我从不知道毛主席还是书法家。至于革命客观条件，我没法预测，但是我希望我能活着看到革命成功。
