市场经济（企业-家庭-均衡）

企业的行为

假设市场存在大量企业，且它们都是同质的（即生产的商品是一样的，所以这些企业可以由一个企业代表所有），生产函数为：$Y(t)=F(K(t), A(t) L(t))$，$A(t)$ 外生给定，与solow模型中一样，其增长率为g。生产函数为一齐次函数。企业是价格接受者，在要素市场租用资本和雇佣工人，在产品市场销售生产的商品。

假设企业的股份被家庭拥有，故企业的利润被家庭所占有。将生产的商品的价格单位化，即令$P(t) \equiv 1$ ，企业租用资本的利率为 $r(t)$，雇佣工人的工资为 $W(t)$，这两个变量都是外生的。

那么，一个代表性企业的利润最大化的行为由下式给出：

$\max \pi(t)=Y(t)-(r(t)+\delta) K(t)-W(t) L(t)$

将生产函数带入上式得：

$\max \pi(t)=F(K(t)$,

$A(t) L(t))-(r(t)+\delta) K(t)-W(t) L(t)$

一阶求导：

对K：$F_{K}(K(t), A(t) L(t))=r(t)+\delta$；

对L：$F_{L}(K(t), A(t) L(t))=W(t)$

由于$F_{K}(K(t)$, $A(t) L(t))=f^{\prime}(k(t))$

且$F_{L}(K(t)$,

$A(t) L(t))=A\left[f\left(k(t)-k(t) f^{\prime}(k(t))\right)\right]$

那么有:

$f^{\prime}(k(t))=r(t)+\delta$ ;
$w(t)=f(k(t))-k(t) f^{\prime}(k(t))-(1)$

其中, $w(t)=\frac{W(t)}{A(t)}$ 称为单位有效劳动的工资, 资本折旧率 $\delta \in(0,1)$

由齐次函数的欧拉定理得，

$\frac{\partial F(K(t), A(t) L(t))}{\partial K}+\frac{\partial F(K(t), A(t) L(t))}{\partial L}=F(K(t)$,

$A(t) L(t))$

那么利润最大化时企业的利润 $\pi(t)=0$
