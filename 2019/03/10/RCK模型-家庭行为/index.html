RCK模型-家庭行为

发表于 2019-03-10 分类于 学习笔记
本文字数： 1.4k 阅读时长 ≈ 1 分钟

假设市场存在许多家庭，数量为H，且也都是同质家庭，则可由一个代表性的家庭代表所有家庭。家庭在其预算约束下最大化自身效用的行为由下式给出：

$m a x \int _ { t = 0 } ^ { \infty } e ^ { - p t } u ( C ( t ) \frac { L ^ { s } ( t ) } { H } ) d t$

s.t. $C(t) \frac{L^{s}(t)}{H}+\left(\frac{E(t)}{H}\right)^{\prime}=r(t) \frac{E(t)}{H}+W(t) \frac{L^{s}(t)}{H}+\frac{\pi(t)}{H}$

其中，

$u ( C ( t ) ) = \frac { C ( t ) ^ { 1 - 0 } - 1 } { 1 - \theta } , \theta \neq 1 ; u ( C ( t ) ) = \ln C ( t ) , \theta = 1$

这是一个常相对风险规避效用函数（CRRA），规避系数为 $\theta$。$C ( t )$，是单位资本消费，$\rho \in(0, \infty)$ 是时间偏好率，即贴现率。$E(t)$ 是经济中的总的金融资产，初始值为 $E(0)$ 给定, $\pi(t)=0$。

定义 $c(t) \equiv \frac{C(t)}{A(t)}, e(t) \equiv \frac{E(t)}{A(t) L(t)}$ 分别为单位有效劳动的消费，单位有效劳动的金融资产; $B(t) \equiv \frac{A(0)^{1-\theta} L(0)}{H}, \beta \equiv \rho-n-(1-\theta) g$ 可以将上述最优化式子转化为下述简便的形式:

$\max \int_{t=0}^{\infty} e^{-\beta t} B \frac{c(t)^{1-\theta}}{1-\theta} d t$

s.t. $e(t)=[r(t)-n-g] e(t)+w(t)-c(t), \quad e(0)$ 给定

根据第二篇文章中的知识点，c(t)是控制变量，e(t)是状态变量，这个最优化问题是一个无穷边界自治问题，tvc是 $\lim _{t \rightarrow \infty} e(t)=e^{*}$，

定义hamiltonian function：

$H(t, c(t), e(t)$,

$\lambda(t))=e^{-\beta t} B \frac{c(t)^{1-\theta}}{1-\theta}+\lambda(t)[((r(t)-n-g) e(t)+w(t)-c(t)]$ , $\lambda$ 是现值朗格朗日乘子

$\frac{\partial H}{\partial c}=0: e^{-\beta t} B c(t)^{-\theta}=\lambda(t)-(2)$
$\frac{\partial H}{\partial e}=-\lambda^{\prime}(t): \lambda(t)[r(t)-n-g]=-\lambda^{\prime}(t)-(3)$
$\frac{\partial H}{\partial \lambda}=e^{\prime}(t): e^{\prime}(t)=e(t)[r(t)-n-g]+w(t)-c(t)-(4)$

对（2）式两侧取对数，再对t求导，联合（3）式，得：

$\frac{e^{\prime}(t)}{e(t)}=\frac{1}{\theta}[r(t)-(\rho+\theta g)]-(5)$

那么，式子（4）和式子（5）构成了两个非线性动态系统，边界条件是tvc和e(0)。
